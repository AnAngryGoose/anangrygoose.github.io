    Overview

    This is just reference and notes for all my project stuff. Most of this is just for personal reference, but some of it may be useful.

    This has a broad scope, from server setup, filesystems, and networking, to individual services.

    Currently using list below [more or less, may have notes in other pages]

    Infrastructure

    Debian Linux - It's debian

    Ansible - IT Automation. Easily create and manage multiple servers.

    Proxmox - Lightweight, easily managed hypervisor.

    MergerFS - FUSE-based union filesystem. Merge multiple drives to be read and used as one. Lots of options.

    SNAPRaid - SNAPshot Raid. Take a snapshot of a pool (mergerFS), create parity, rebuild if a drive dies.

    Docker - Platform for building, packaging, and distributing containerized applications

    git - version control system

    Network Management

    OPNSense - An open source, easy-to-use and easy-to-build FreeBSD based firewall and routing platform.

    Omada Controller - TP-Link Omada Software Controller (also known as TP-Link Omada Network Application) to centrally manage TP-Link Omada Hardware

    Tailscale - A Zero Trust identity-based connectivity platform. Allows remote, secure access to a defined network (tailnet).

    Adguard Home - A network-wide software for blocking ads & tracking.

    Cloudflare - DNS server, domain registrar, CDN. Primarily used here for tunnels.

    Cloud Services & File sharing

    Nextcloud - A suite of client-server software for creating and using file hosting services

    Immich - Self-hosted photo and video management solution

    Filebrowser Quantum - Web-based file manager with configurable sources, OIDC authentication, Office file support, and real-time indexing.

    Media Server

    Plex Media Server - Media server for streaming content

    Jellyfin - Media server for streaming content. Less convoluted and more privacy focused than Plex.

    Arr stack - arr software for media management. Radarr, sonarr, etc.

    Smart Home

    Home Assistant - Home Assistant is free and open-source software used to enable centralized home automation.

    MQTT - Message Queuing Telemetry Transport - it is a lightweight, publish-subscribe network protocol designed for low-bandwidth, unreliable, or high-latency networks.

    Generative AI

    Ollama - Lightweight, extensible framework for building and running language models on the local machine. It provides a simple API for creating, running, and managing models, as well as a library of pre-built models that can be easily used in a variety of applications.

    Utilities

    convertx - Self hosted, web based file convertor with support for A LOT of file types.

    BentoPDF - Web based PDF editor. Does all work in browser.

    ittools - Collection of handy online tools for developers, with great UX.

    drawio - self hosted version of drawio. draw.io is a JavaScript, client-side editor for general diagramming.
